## üìñ Key Concepts

### 5-Layer Perception Toolkit

A structured approach to extracting wisdom from personal stories:

1. **Narrative Layer**: Story vs legend distinction
2. **Somatic Layer**: Body sensations and emotions
3. **Attention Layer**: Tiny details and observations
4. **Synesthetic Layer**: Non-visual sensory experiences
5. **Temporal Layer**: Sound and rhythm of the moment

### Z-Protocol v2.0

Ethical framework for AI training data:

- **5-Tier Classification**: Public, Personal, Cultural, Sacred, Therapeutic
- **8 Consent Types**: Training, Commercial, Research, Attribution, etc.
- **Revenue Sharing**: 15-30% based on content tier
- **Cultural Protection**: Community attribution and benefit sharing

### Agent Orchestration

- **Y Agent (Gemini)**: Distillation & essence extraction
- **X Agent (Perplexity)**: Quality analysis & validation
- **Z Agent**: Ethics & protocol validation
- **P Agent**: Pedagogical documentation
- **XV Agent**: Experimental validation
